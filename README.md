## Stack
Built on the MERN Stack on the frontend and firebase for Authentication and Database storage.


## Getting Started

- First, Install the dependencies:

```bash
npm install
# or
yarn install
```
- Create a new project on FireBase and copy the product key and details
- Paste the details in the `firbase.js` file in the project

- Run the development server:

```bash
yarn start
```

Open [http://localhost:3001](http://localhost:3001) with your browser to see the result.
